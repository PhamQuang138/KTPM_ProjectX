07 · Tin cậy & vận hành

Bảo vệ dữ liệu từ request
đến quyền sở hữu nội dung.

AUTH

Xác thực và phân quyền

JWT, mật khẩu bcrypt, USER/ADMIN, route bảo vệ và kiểm tra owner trước thao tác sửa/xóa.

API

Biên API được kiểm soát

Zod validation, Helmet, CORS allow-list, rate limit production và giới hạn JSON 5 MB.

MEDIA

Ảnh tách khỏi database

Local lưu trong uploads; production dùng Vercel Blob. PostgreSQL chỉ giữ URL và metadata.
