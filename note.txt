CARHUB - GHI CHU KIEN TRUC VA VAN HANH
Cap nhat: 13/06/2026

1. TONG QUAN

CarHub Garage la nen tang full-stack ket hop cong dong oto, Garage ca nhan,
Marketplace, noi dung chuyen de, nhan tin va tro ly tim xe bang AI.

Du an la monorepo npm workspaces gom:
- frontend: React 19 + Vite + TypeScript + Tailwind CSS + Zustand + Motion.
- backend: Node.js 20+ + Express + TypeScript + Prisma ORM.
- database: PostgreSQL, hien tai phu hop voi Neon cloud.
- AI: OpenRouter voi model mac dinh nex-agi/nex-n2-pro:free, chi goi tu backend.
- luu anh production: Vercel Blob; database chi luu URL va metadata.
- deploy: Vercel multi-service, frontend tai "/" va backend tai "/api".

3. CAC NHOM DU LIEU TRONG POSTGRESQL

He thong hien co 19 model, chia theo cac mien:

Identity:
- User: tai khoan, mat khau bcrypt, avatar, banner, profile, role, verified.
- UserFollow, UserRating: theo doi va danh gia thanh vien.
- PasswordResetToken: ma dat lai mat khau, thoi han va trang thai da dung.

Community:
- Post + Image: bai cong dong va nhieu anh.
- PostLike, PostComment, PostBookmark, PostShare: tuong tac bai viet.
- Article: bai viet chuyen de, trang thai DRAFT/PUBLISHED.

Garage va Marketplace:
- GarageVehicle: xe ca nhan, thong so va danh sach URL anh.
- VehicleListing: tin dang ban, co the lien ket GarageVehicle.
- ListingFavorite, ListingComment: luu va binh luan tin xe.
- Vehicle: model listing cu van con trong schema de tuong thich du lieu.

Engagement:
- Conversation + Message: nhan tin truc tiep hoac theo mot tin xe.
- Notification: thong bao like, comment, follow, marketplace va message.

4. LUONG HOAT DONG CHINH

Tai khoan va ho so:
- Dang ky, dang nhap, dang xuat tai /api/auth.
- Quen mat khau tao ma co thoi han va co the gui qua SMTP.
- JWT duoc frontend luu trong localStorage va gui bang Authorization Bearer.
- Thanh vien cap nhat bio, location, focusBrands, avatar va banner.
- Co the tim tai khoan, xem profile, theo doi/bo theo doi va danh gia thanh vien.
- USER quan ly noi dung cua minh; ADMIN co quyen quan tri toan he thong.
- ADMIN va tai khoan verified hien BadgeCheck mau xanh.

Feed cong dong:
- Tao post gom caption va toi da 8 anh.
- Tac gia hoac ADMIN co the xoa post.
- Khi sua post, API chi cap nhat content/caption; anh khong bi sua kem.
- Ho tro like, comment, bookmark, share va danh sach da thich/da luu.
- Bai viet co the duoc chia se vao mot cuoc tro chuyen.
- Feed uu tien bai cua ADMIN va tai khoan verified.

Garage va cho xe:
- GarageVehicle la xe ca nhan; VehicleListing la tin dang ban rieng.
- Co the them, sua, xoa xe; xoa xe se xoa listing lien quan trong transaction.
- Mot xe co toi da 12 anh; anh dau tien la anh bia.
- Co the tao listing tu Garage, sua, danh dau da ban va xoa listing.
- Marketplace co search, filter, sort va phan trang 10 tin/trang.
- Bo loc gom category, condition, make, bodyType, fuelType, transmission va nam.
- Sort ho tro newest, trending va oldest.
- Thanh vien co the luu tin, binh luan va danh gia nguoi ban.

Nhan tin va thong bao:
- Co the tao conversation truc tiep giua hai thanh vien.
- Nhan "Lien he" tao/mo conversation gan voi listing dang quan tam.
- Moi conversation tai toi da 200 message va danh dau message da doc khi mo.
- Frontend polling message va danh sach conversation moi 5 giay khi dock dang mo.
- Gui message, like/comment/follow va tuong tac marketplace co the tao Notification.
- Nguoi dung xem toi da 50 thong bao moi nhat va danh dau tat ca da doc.

Tro ly tim xe AI:
- Chatbot chi hien tren frontend khi nguoi dung da dang nhap.
- Route backend POST /api/ai/chat co requireAuth de bao ve quota OpenRouter.
- Nguoi dung tim xe bang cau hoi tieng Viet, anh, hoac ket hop ca hai.
- Frontend nen anh ve WEBP, toi da 1920 px truoc khi upload.
- Backend dung OpenRouter structured output de tach hang, mau, kieu xe, nam, gia,
  dia diem, nhien lieu, hop so, tu khoa va so luong mong muon.
- Neu gia duoc nhap bang VND, prompt yeu cau model quy doi gan dung sang USD.
- Backend truy van listing that trong PostgreSQL va xep hang theo bo loc,
  tu khoa, favorite va comment; AI khong tu tao listing.
- Toi da 20 ket qua; mac dinh 6 ket qua.
- Anh AI chi duoc tai tu localhost, PUBLIC_BACKEND_URL hoac Vercel Blob;
  chi chap nhan JPEG, PNG, WEBP va toi da 4 MB o backend.

Quan tri:
- Dashboard thong ke user, post, garage vehicle, listing, article, comment,
  rating, follow va trang thai draft/published.
- Quan ly va xoa user; bat/tat verified professional.
- Duyet trang thai va xoa post.
- Cap nhat trang thai/xoa listing va GarageVehicle.
- Cap nhat trang thai/xoa article.
- Xoa comment, rating va quan he follow.
- Khong cho xoa admin cuoi cung.

Anh:
- Frontend gui file qua /api/uploads/images.
- Frontend chap nhan JPG/JPEG/PNG/WEBP toi da 8 MB truoc khi nen.
- Backend nhan file da xu ly toi da 4 MB bang Multer memory storage.
- Local development luu file trong uploads/images va Express phuc vu /uploads.
- Production hoac khi dung database remote bat buoc co BLOB_READ_WRITE_TOKEN.
- Database khong luu binary/base64, chi luu URL va metadata.

5. CHAY LOCAL

Yeu cau:
- Node.js 20 tro len.
- PostgreSQL/Neon co the truy cap.
- Tao backend/.env tu backend/.env.example.

Lenh tai thu muc goc:

  npm install
  npm run db:generate --workspace backend
  npm run db:migrate:deploy --workspace backend
  npm run dev

Dia chi:
- Frontend: http://localhost:3000
- Backend health: http://localhost:4000/api/health

Chay rieng:

  npm run dev:frontend
  npm run dev:backend

Kiem tra va build:

  npm run lint
  npm run build

Database:

  npm run db:generate --workspace backend
  npm run db:migrate:deploy --workspace backend
  npm run db:seed --workspace backend
  npm run db:seed-daily-content --workspace backend
  npm run db:ensure-admin --workspace backend
  npm run db:studio --workspace backend

Luu y: db:seed co the reset du lieu ung dung, khong chay tren database dung
chung neu chua thong nhat.

db:seed-daily-content tao lai 45 bai cong dong khong trung tieu de va 100 tin
cho xe pho thong. Bai cong dong seed khong gan anh tuy tien. Tin xe dung anh
dai dien dung dong xe tu Wikipedia/Wikimedia Commons, co ghi ro nguon va canh
bao day la du lieu demo. Script xoa cac seed cu va tin cua tai khoan
@carhub-demo.com truoc khi tao lai; khong xoa tin cua tai khoan nguoi dung.

Cai dat nguoi dung:

- Trang /settings luu tuy chon theo tai khoan trong PostgreSQL.
- Ho tro giao dien sang/toi/theo he thong, mat do hien thi va co chu.
- Co the tu dong mo tro ly AI mot lan trong moi phien trinh duyet.
- Co the bat/tat thong bao cong dong, cho xe va tin nhan rieng.
- Khi thay doi schema, chay db:migrate:deploy va db:generate truoc khi build.

Chatbot tim xe:

- Gia va bo loc ngan sach su dung VND qua minPriceVnd/maxPriceVnd.
- Khi hoi "gia tot", "gia hop ly" hoac "dang tien" ma chua neu ngan sach,
  chatbot tham khao khoang 500 trieu den 1 ty dong.
- Frontend gui toi da 10 luot hoi dap gan nhat de chatbot hieu cau noi tiep.
- Bo loc hang, kieu xe, nhien lieu, hop so, nam va gia la dieu kien bat buoc;
  diem xep hang khong duoc phep dua xe sai bo loc vao ket qua.

6. BIEN MOI TRUONG BACKEND QUAN TRONG

Core:
- NODE_ENV=development hoac production
- PORT=4000, chi can cho local
- DATABASE_URL=PostgreSQL/Neon connection string
- CORS_ORIGIN=danh sach origin, phan tach bang dau phay
- PUBLIC_BACKEND_URL=URL public de AI cho phep tai anh tu backend
- JWT_SECRET=chuoi bi mat it nhat 32 ky tu
- JWT_EXPIRES_IN=7d
- ADMIN_USERNAME va ADMIN_PASSWORD
- RATE_LIMIT_WINDOW_MS va RATE_LIMIT_MAX
- LOG_LEVEL

AI:
- OPENROUTER_API_KEY=key tu OpenRouter
- OPENROUTER_MODEL=nex-agi/nex-n2-pro:free hoac model duoc cau hinh
- OPENROUTER_SITE_URL=URL cua ung dung, dung cho HTTP-Referer
- OPENROUTER_APP_NAME=ten ung dung hien tren OpenRouter

Media:
- BLOB_READ_WRITE_TOKEN, bat buoc tren production/database remote

Email dat lai mat khau:
- SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS, SMTP_FROM

Khong commit backend/.env, token Blob, OpenRouter key, JWT secret, SMTP password
hoac mat khau database.

Frontend:
- Local co the dung VITE_API_URL=/api nho Vite proxy.
- Vercel multi-service dung API cung domain; api.ts tu xu ly prefix production.

8. BAO MAT VA RANG BUOC

- Helmet dat security headers.
- CORS chi cho phep origin duoc cau hinh hoac Vercel origin hop le.
- Rate limit bat tren production; bo qua OPTIONS va health check.
- JSON body gioi han 5 MB.
- Zod validate cac body quan trong truoc controller.
- Route sua/xoa phai kiem tra owner hoac role ADMIN.
- Conversation va message chi tra cho participant.
- Khong cho tao conversation voi chinh minh.
- Password duoc hash bang bcrypt; reset code duoc hash va co expiresAt.
- Upload kiem tra ca MIME type, extension va kich thuoc.
- AI image URL co allow-list host, protocol, MIME type va timeout.
- Khong ghi key, token hoac mat khau vao log.

10. NGUYEN TAC KHI PHAT TRIEN TIEP

- Them model: sua schema.prisma, tao migration, sau do prisma generate.
- Them API theo luong route -> validation -> controller -> service -> Prisma.
- Khong luu file anh vao PostgreSQL.
- Khong goi OpenRouter truc tiep tu frontend va khong lo OPENROUTER_API_KEY.
- Moi route sua/xoa phai kiem tra owner hoac role ADMIN.
- Dung SocialPost cho feed de dong bo like/comment/bookmark/edit/delete.
- Tao notification qua notificationService va tranh gui cho chinh actor.
- Thao tac lien quan nhieu bang nen dung Prisma transaction.
- Khi sidebar dong/mo, main content phai doi ca margin-left va width.
- Chay npm run lint va npm run build truoc khi merge.
