# Chỉnh thương hiệu CarHub

- Logo trên giao diện: thay file `public/brand/carhub-logo.jpg`.
- Icon trên tab/URL: thay file `public/brand/carhub-favicon.jpg`.
- Component hiển thị logo: `src/components/BrandLogo.tsx`.
- Font toàn website: sửa `--font-sans` và `--font-display` trong `src/index.css`.
- Tiêu đề lớn trang Khám phá: sửa `.hero-title` và `.hero-title-accent`
  trong `src/index.css`.
- Nội dung chữ “MUA BÁN...” nằm trong `src/pages/Home.tsx`.

Giữ nguyên tên hai file ảnh để không phải sửa lại code.
