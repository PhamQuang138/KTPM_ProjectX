# KTPM_ProjectX
Trong bối cảnh nhu cầu mua bán và tìm hiểu về xe ngày càng tăng, việc xây dựng một nền tảng tích hợp giữa thương mại điện tử và chia sẻ kiến thức là cần thiết.

Đề tài này nhằm xây dựng một hệ thống website cho phép người dùng:

Đăng bán và tìm kiếm các loại xe
Xem thông tin chi tiết, so sánh xe
Viết và đọc các bài blog liên quan đến xe (đánh giá, kinh nghiệm, tin tức)
Tương tác với cộng đồng thông qua bình luận và đánh giá

Hệ thống được thiết kế theo hướng ứng dụng web hiện đại, đảm bảo:

Quản lý người dùng và phân quyền (user/admin)
Xử lý dữ liệu hiệu quả
Giao diện trực quan, dễ sử dụng
🔹 Mục tiêu
Xây dựng hệ thống web hoàn chỉnh gồm frontend và backend
Áp dụng các nguyên lý của Kỹ thuật phần mềm:
Phân tích yêu cầu (SRS)
Thiết kế hệ thống (UML)
Xây dựng và kiểm thử
